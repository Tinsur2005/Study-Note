# 面向对象基础：类和对象、封装、构造方法

面向对象编程OOP：Object Oriented Programming；

面向对象的特征：抽象、封装、继承、多态。

## 一、类和对象

### 1、介绍

类（Class）：对象是世界中事物的抽象，抽象模板、图纸，描述一类事物`共同属性（成员变量）+ 行为（成员方法）`；

对象（Object）：现实世界中具体的个体，类的`实例`，根据模板创建出来真实个体。

>**类和对象的联系：**
>
>1. 现实世界都是由很多对象组成的，基于对象的共同特征抽象出类。
>
>2. 对象：真实存在的对象
>
>3. 类中包含：
>   - 所有对象所共有的特征：属性（静）
>   - 所有对象所共有的行为：方法（动）
>
>4. 类是对象的模板，对象是类的具体实例。
>
>5. 一个类可以创建多个对象, 同一个类的不同对象，结构相同，数据不同。

类的命名规范：采用`大驼峰（Pascal）`命名法，如 Student、User。

定义一个实体类（每个实体类和数据库中的表一一对应）：

```java
public class Student {
    //属性
    int id;
    String name;
    int age;
    String gender;
}

public class OOPDemo {
    @Test
    public void test1(){
        Student student1= new Student();
        //对象.属性
        student1.id = 1;
        student1.name = "张三";
        student1.age = 18;
        student1.gender = "男";
        System.out.println(student1.name);

        Student student2= new Student();
        student2.id = 1;
        student2.name = "李四";
        student2.age = 19;
        student2.gender = "女";
        System.out.println(student2.name);
    }
}
```

### 2、对象的创建过程

**堆：**new出来`对象`放在堆里面；

**栈：**`变量（引用）`放在栈里面，栈中存储的是对象的`地址值；`；

**方法区：**存放和加载的类的信息，在用到一个类的时候，需要把类的信息放到方法堆里。

创建对象有两种方法：

```java
// 方式1：引用变量接收对象
Student student = new Student();

// 方式2：匿名对象（没有引用，只能立即使用一次）
new Student().show();
```

对象的创建过程：

1. `new` 在堆内存开辟空间
2. 成员变量默认初始化
3. 执行构造方法，有关构造方法的内容下文会提到
4. 返回对象地址给引用变量

student是`栈`中的`引用变量`，保存堆中对象的`地址值`，多个引用可以指向同一个对象：

```java
Student s2 = s1; // s1、s2指向同一个堆对象
```

### 3、数据类型的默认值

在前面记录过一些基本数据类型的默认值 [1-Java介绍.md](1-Java介绍.md) ，这里展开来写：

int类型默认值是：0

double类型默认值是：0.0

boolean类型默认值是：false

**String类型默认值是：**null

**int[]类型默认值是：**null

**Student[]类型默认值是：**null

**boolean[]类型默认值是：**null

**总结可得到规律：**所有`引用数据类型`的默认值都是`null`。

**举例：**

```java
int[] array = new int[3]; //里面的存放默认值是0
boolean[] array = new boolean[3]; //里面的存放默认值是false
Student[] array = new Student[3]; //里面的存放默认值是null
```

### 4、空指针异常

**空指针异常（引用数据类型）：**NullPointerException

如下即会报空指针异常：

```java
@Test
public void test4() {
    //NullPointerException空指针异常（引用数据类型）
    Student student = null;
    System.out.println(student);//null
    // null.方法   null.属性
    System.out.println(student.id);//会报NullPointerException空指针异常
}
```

在Student是null时，直接调用`System.out.println(Student);`不会报错，但此时调用Student类的属性等就会报错。

## 二、toString

`toString()` 是Object类中定义的实例方法，所有Java对象都继承该方法，作用是返回该对象的`字符串表示形式`。

Object默认实现（不重写时）可读性极差，无法看到对象内部属性，因此实体类建议重写。

IDEA快速重写toString快捷键：`Alt + Insert` → `toString()`。

```java
@Override
public String toString() {
    return "Student{" +
        "id=" + id +
        ", name='" + name + '\'' +
        ", age=" + age +
        ", gender='" + gender + '\'' +
        '}';
}
```

在需要打印对象的属性时，直接打印对象名即默认调用对象的toString方法：

```java
Student student1= new Student();
//对象.属性
student1.id = 1;
student1.name = "张三";
student1.age = 18;
student1.gender = "男";
System.out.println(student1);
```

输出：

```bash
Student{id=1, name='张三', age=18, gender='男'}
```

## 三、封装

面向对象的封装是一种编程概念和原则，它通过将数据和操作封装在一个对象中，以实现信息隐藏和安全性。

### 1、Get和Set方法

封装使得对象的内部细节对外部不可见，只暴露必要的接口供其他对象进行交互。

例如以下这个类中的各属性均为`private`，被`private`修饰的属性只能在本类中访问：

```java
private int id;//private私有的：只能在类的内部访问
private String name;
private int age;
private String gender;
```

所以要使用`get/set`方法来提供外部访问的工具，IDEA快速重写toString快捷键是`Alt + Insert` → `getter & setter`：

```java
    public int getId() {
        return id;
    }

    public void setId(int id) {
        this.id = id;
    }
```

:warning: 需要注意是，在setId中使用的`this`，代表当前对象，用于区分属性和参数，若写成`id = id`，根据Java的就近原则，两个`id`均指向参数。

另外，也可以在get和set方法中实现权限的控制，如：

```java
public void setAge(int age) {
    if (age > 0 && age < 250) {
        this.age = age;
    }
}
```

调用：

```java
//stu.id = 1;//无法赋值
stu.setId(5);
System.out.println(stu.getId());
```

### 2、访问修饰符

private（私有的）：仅当前类内部可用。

default（默认权限，不写任何修饰符，也叫包访问权限）：同一个包下所有类可以访问，跨包不能访问。

protected（受保护的）：同包所有类 + 不同包中的子类。

public（公共的）：任何地方都能访问

各修饰符访问权限对比：

| 修饰符         | 本类 | 同包 | 子类 | 其他包 |
| -------------- | ---- | ---- | ---- | ------ |
| private        | ✅    | ❌    | ❌    | ❌      |
| default (默认) | ✅    | ✅    | ❌    | ❌      |
| protected      | ✅    | ✅    | ✅    | ❌      |
| public         | ✅    | ✅    | ✅    | ✅      |

### 3、常量（final）

`private final` 修饰变量，只能在构造方法赋值，没有setter

```java
private final String nation = "中国";
public String getNation(){
    return nation;
}
```

## 四、构造方法

构造方法是`创建对象`（new 对象）时自动调用的特殊方法，用于初始化对象成员变量。

构造方法的方法名必须和类名`完全一致`，没有返回值，且方法签名中`不写void`，构造方法`不能被继承和重写`。

构造方法不能手动调用，使用 `new 类名()` 创建对象时自动执行；一个类可以有多个构造方法，即存在构造方法的重载。

构造方法分为`有参构造方法`和`无参构造方法`，顾名思义，有参构造是带有参数的构造方法，无参构造是没有任何参数的构造方法。

:warning:需要注意的是，如果类中没有写任何构造方法，编译器自动生成一个`默认无参构造`；一旦手写了任意一个有参构造，编译器不再提供默认无参构造，所以在手写有参构造的时候，要注意加上一个无参构造，以防止调用无参构造时出现问题。

```java
//无参构造方法
public Student() {
    System.out.println("Student.Student()");
}

//有参构造方法
public Student(int id, String name, int age, String gender) {
    this.id = id;
    this.name = name;
    this.age = age;
    this.gender = gender;
}
```
