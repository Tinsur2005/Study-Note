# 常见的问题的解决方法

## 一、IDEA相关问题

### 1、@Test单元测试时无法输入

Help-Edit Custom VM Options...

新增一行 -> `-Deditable.java.test.console=true` ，重启IDEA后即可解决

## 二、UML图相关

### 1、接口的画法配置修改

StarUML软件显示接口默认是一个圆圈，需要按照下面图示的步骤更改设置：

![3a53d788-55b2-4a0a-9df5-7fadb8624b1a](./../assets/%E5%B8%B8%E8%A7%81%E9%97%AE%E9%A2%98%E7%9A%84%E8%A7%A3%E5%86%B3%E6%96%B9%E6%B3%95/3a53d788-55b2-4a0a-9df5-7fadb8624b1a.png)
