## 一、if

### 1、if-else

```java
if (boolean) {
    代码块
} else if (boolean) {
    代码块
} else if (boolean) {
    代码块
} else { // 默认情况
    代码块
}
```

### 2、三目运算符

**语法：**`boolean ? 数1 ： 数2`

执行过程：计算boolean的值

​               **若为true：**则整个结果为数1

​               **若为false：**则整个结果为数2

```java
	int max1 = num1 >= num2 ? num1 : num2;
    System.out.println(max1);
```

### 3、+号运算符

- 两边都是数字，则做的就是`加法运算`

- 若一边为字符串，则做的是`字符串的拼接`

```java
    public void test3 (){
            int age = 28;
            //豆豆加加
            System.out.println("我的年龄是23岁");
            System.out.println("我的年龄是" + age + "岁");
            String str = "" ;//空字符串
            //char ch = '';//没有空字符
            System.out.println(10 + 20 + "" + 30);//"3030"
            System.out.println("" + 10 + 20 + 30);//"102030"
    }
```

## 二、循环

### 1、While和For

任何复杂的程序逻辑都可以通过“顺序”、“分支”、“循环”这三种基本结构来实现。

循环的三要素是：`循环条件的初始化`、`循环条件的控制`、`循环条件的改变`

**While条件循环**的用法：

```java
    public void test4(){
        int i = 1;
        while (i <= 5){
            System.out.println("HelloTinsur");
            i++;//执行下一次
        }
```

**For循环**的用法：

```java
	for(变量声明和赋值;循环条件;变量自加){
        循环体
    }
```

```java
    @Test
    public void test5(){
        for (int i = 1; i <= 5; i++) {
            System.out.println("HelloTinsur");
        }
    }
```



### 2、continue、break

**continue：**跳出本次`循环`，继续下一次循环

**break：**跳出离他最近的那层`循环`
