# 数组

## 一、数组的基本概念

### 1、算法和数据结构

程序 = 算法 + 数据结构

其中，算法指`解决程序的流程步骤`，数据结构指`将数据按照某种特定的结构来存储`，设计良好的数据结构会导致良好的算法。

### 2、数组的介绍

数组就是最简单的数据结构，数组是存放`同一种类型`数据的集合，在内存里面是开辟一块`连续的区域`；

在Java中，数组是一个`引用数据类型`；

定义数组的基本方法：`元素类型[] 数组名 = new 元素类型[长度] ;`，其他方法见下方章节中的`5、数组的定义方式`；

数组的访问方法：`数组名[下标]`。

**示例：**

```java
    int num = 3;
    int[]  array = new int[4];
	//1、存放整数类型的数组
	//2、长度是4 （数组缺点长度固定，ArrayList解决长度固定的问题）
	//3、只能存放int类型
```

### 3、数组的遍历

在IDEA环境中，可以使用``快捷键快速创建遍历循环：

`数组名.for`：增强的for循环，不需要遍历下标；

```java
    for (int item : array) {
        System.out.println(item);
    }
```

`数组名.fori`：使用for循环遍历，需要遍历下标。

**正序遍历：**

```java
    @Test
    public void test1() {
        int[] array = new int[4];
        array[0] = 23;
        array[1] = 3;
        array[2] = 4;
        array[3] = 5;
        for (int i = 0; i < array.length; i++) {
            System.out.println(array[i]);
        }
    }
```

**倒序遍历：**

```java
        for (int i = array.length; i >= 0 ; i--) {
            System.out.println(array[i]);
        }
```

### 4、数组成员的默认值

**int类型：**0

**double/flaot类型：**0.0

**boolean类型：**false

如果是**类**的话，则默认值是null，这个在后面的章节里会提到。

### 5、数组的定义方式

**静态初始化：**在声明数组的同时，直接给出数组元素的初始值。适用于数组元素已知的情况，这种情况先，赋值有几个元素，则数组的长度就是几。

```java
int[] array = {32,5,7,87};

int[] array = new int[]{32,5,7,87};
```

**动态初始化：**在声明数组时，仅指定数组的长度，然后通过赋值语句逐个给数组元素赋值。适用于数组元素在声明时未知的情况，上文中`2.数字的介绍`中提到的就是这种情况。

```java
int[] array = new int[4];

array[0] = 34;
```

### 6、数组下标越界异常

**数组下标越界异常：**java.lang.ArrayIndexOutOfBoundsException: x（最后的数字表示越界的数）

## 二、数组的累加和、最大/小值、排序

### 1、数组的累加和

```java
    @Test
    public void test189() {
        int[] array = {11, 2, 45, 15};
        //array.fori   fori
        int sum = 0;
        for (int i = 0; i < array.length; i++) {
            sum = sum + array[i];
        }
        System.out.println("sum: " + sum);
    }
```

### 2、数组的最大值

```java
    @Test
    public void test2() {
        int[] arr1 = {33, 2, 45, 19};
        int max = arr1[0];
        for (int i = 1; i < arr1.length; i++) {
            // i = 0 时第一步会和自己比较，没有意义，所以从i=1开始比较
            if (arr1[i] > max) {
                max = arr1[i];
            }
        }
        System.out.println(max);
    }
```

### 3、数组的最小值

```java
    @Test
    public void test3(){

        int min = arr2[0];
        for (int i = 1; i < arr2.length; i++) {
            if (arr2[i] < min){
                min = arr2[i];
            }

        System.out.println(min);
    }
```

### 4、数组的排序：冒泡排序

```java
	@Test
    //冒泡排序
    public void test4(){
        int[] array = {40, 17, 21, 1};
        sort(array);
        for (int i : array) {
            System.out.println(i);
        }
    }
    //---方法签名-----//
    private void sort(int[] array) {
        for (int i = 1; i < array.length; i++) {
            //i表示第几轮
            for (int j = 0; j < array.length - i ; j++) {
                if(array[j] > array[j+1]){
                    int temp = array[j];
                    array[j] = array[j+1];
                    array[j+1] = temp;
                }
            }
        }
    }
```
