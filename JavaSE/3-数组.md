# 数组

## 一、数组的基本概念

### 1、算法和数据结构

程序 = 算法 + 数据结构

其中，算法指`解决程序的流程步骤`，数据结构指`将数据按照某种特定的结构来存储`，设计良好的数据结构会导致良好的算法。

### 2、数组的介绍

数组就是最简单的数据结构，数组是存放`同一种类型`数据的集合，在内存里面是开辟一块`连续的区域`；

在Java中，数组是一个`引用数据类型`；

定义数组的方法：`元素类型[] 数组名 = new 元素类型[长度] ;`。

**示例：**

```java
    int num = 3;
    int[]  array = new int[4];
	//1、存放整数类型的数组
	//2、长度是4 （数组缺点长度固定，ArrayList解决长度固定的问题）
	//3、只能存放int类型
```
